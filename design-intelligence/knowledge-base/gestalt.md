# Gestalt Principles · 格式塔心理学原则

> 人们如何感知视觉元素之间关系的心理学原理。

## 1. 邻近性 (Proximity)

**原理**：空间上靠近的元素被视为一组。

**应用**：
- 卡片间距 < 卡片内 padding，确保卡片之间视觉分离
- 导航项目间距一致
- 图注离图片越近越好

```css
/* ✅ 正确：卡片间距 < 内部 padding */
.card { padding: 24px; }
.grid { gap: 16px; }

/* ❌ 错误：卡片间距 > 内部 padding，分组不清晰 */
.card { padding: 12px; }
.grid { gap: 24px; }
```

## 2. 相似性 (Similarity)

**原理**：看起来相似的元素被视为同一类型。

**应用**：
- 同一层级字号/颜色/字重一致
- 所有链接同色
- CTA 按钮统一风格

## 3. 闭合性 (Closure)

**原理**：大脑自动补全不完整的图形。

**应用**：
- Logo 和图标可简化
- 背景线可微不可见
- 留白可形成图形

## 4. 连续性 (Continuity)

**原理**：视线沿对齐方向流动。

**应用**：
- 文字左对齐引导阅读
- 图片基线对齐
- 水平对齐比垂直对齐更重要

## 5. 图形-背景 (Figure-Ground)

**原理**：元素分为前景（焦点）和背景（场域）。

**应用**：
- 毛玻璃面板 = figure 突出于 ground
- 阴影高度决定层级
- 颜色反差强化 figure-ground

## 6. 共同区域 (Common Region)

**原理**：同一封闭区域内元素被视为一组。

**应用**：
- Card 和容器用背景色 + 边框 + 阴影
- Section 用不同背景色区分
- Modal/Dialog 用遮罩强化区域
