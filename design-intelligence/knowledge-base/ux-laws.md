# UX Laws · 用户体验法则

> HCI 研究与认知心理学在界面设计中的应用。

## 1. Fitts's Law

**公式**：`Time = a + b × log₂(D / W + 1)`
（D = 到目标的距离，W = 目标宽度）

**关键要点**：
- 目标越大 + 距离越近 = 越快
- 屏幕边缘和角落是"无限大"目标

**设计应用**：
- 导航点 / 箭头 / 按钮触控目标 ≥ 44×44px
- 重要操作放在边缘/角落（如关闭按钮）
- 使用 `::after` 扩展不可见点击区
- 菜单项间距不宜过密

```css
/* ✅ Fitts's Law 安全点击区 */
.nav-dot::after {
  content: '';
  position: absolute;
  inset: calc(-1 * (44px - 100%) / 2);
}
```

## 2. Hick's Law

**公式**：`Time = a + b × log₂(n)`
（n = 选择数量）

**关键要点**：
- 选择越少，决策越快
- 排序/分组减少感知数量

**设计应用**：
- 导航层级 ≤ 3 层
- 菜单项 ≤ 7 项
- 卡片每页 ≤ 4-6 张
- 用分组/分类减少感知数量

## 3. Miller's Law

**公式**：工作记忆容量 ≈ 7 ± 2 组块

**关键要点**：
- 人们一次能记住 5-9 个组块
- 组块（chunking）可扩展容量

**设计应用**：
- List 布局每页 4-6 项
- Gallery 每页 4-8 张
- Tab 导航 ≤ 5 项
- 用分类/分层组织信息

## 4. Jakob's Law

**原理**：用户期望在网站上使用与其他网站相同的交互模式。

**设计应用**：
- 使用标准导航模式（上左右）
- 遵循平台惯例（Mac/Windows/Web）
- 翻页方向：左/上为上一页，右/下为下一页
- 快捷键符合常见映射

## 6. Aesthetic-Usability Effect

**原理**：美观的设计被视为更易用。

**应用**：
- 美观 ≠ 可用，但美观增加容错率
- 精致排版提升内容信任度
- 统一主题色 vs 随意配色
