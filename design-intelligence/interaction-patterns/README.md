# Interaction Pattern Library

> 按场景选择交互模式，不要随机添加动效。

## 索引

### Page Enter 页面进入
| 模式 | 适用风格 | 说明 |
|------|---------|------|
| Fade In | 所有 | 通用淡入 |
| Scale In | Minimal, Editorial | 缩放进入 |
| Stagger | Bento, Gallery | 卡片逐个入场 |
| Reveal | Editorial, Architectural | 图片裁切入场 |
| Split Text | Luxury, Editorial | 文字逐个显现 |
| Glitch | Cyberpunk | 故障效果入场 |
