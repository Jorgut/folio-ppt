# Minimal · 极简主义

> 通用 · 商业 · 科技

## 语感

"少即是多" — 大量留白、克制排版、去除一切装饰。

## 参考品牌

Apple, Muji, Braun, Vitsoe

## 字体

```
标题: Helvetica Neue Light / Inter Light (字重 300)
正文: Helvetica Neue / Inter (字重 400)
元数据: SF Mono / JetBrains Mono (字重 400)
```

## 颜色

| 角色 | 浅色 | 深色 |
|------|------|------|
| 背景 | `#ffffff` | `#000000` |
| 文字 | `#1a1a1a` | `#f5f5f5` |
| 强调 | `#4a4a4a` | `#b0b0b0` |
| 次要 | `#808080` | `#888888` |

## 间距

大量使用 8pt Grid 的上限：`--sp-8` (64px) / `--sp-9` (80px)

## 动效

L2 标准 — FadeIn + SlideUp，无 Decorative 动效

## 特效

无 — 去除所有装饰性特效

## 布局

- Cover: 居中标题 + 极大字
- Split: 4/8 或 3/9
- 大量留白，内容区域 ≤ 60%

## 禁忌

- ❌ 使用任何装饰元素（花纹、渐变、噪点）
- ❌ 多色强调（只用单色或灰度）
- ❌ 过度动效（仅限翻页过渡）
- ❌ 全出血图（破坏留白）
