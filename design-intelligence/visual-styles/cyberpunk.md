# Cyberpunk · 赛博朋克

> 创意 · 游戏 · 未来

## 语感

"霓虹夜色" — 暗底 + 霓虹发光线、色差效果、像素/扫描线装饰、未来科技感。

## 参考品牌

Cyberpunk 2077, Blade Runner, Tron, Ghost in the Shell

## 字体

```
标题: Orbitron / Rajdhani Bold (字重 700) — 全大写
正文: Rajdhani / Exo 2 (字重 400)
数字: Orbitron / JetBrains Mono (字重 600)
元数据: 紧凑无衬线 + letter-spacing: 0.05em
```

## 颜色

| 角色 | 深色 |
|------|------|
| 背景 | `#0a0a1a` |
| 面板 | `#12122a` |
| 文字 | `#e8e8ff` |
| 强调 | `#00f0ff` 青 / `#ff00aa` 粉 / `#ffcc00` 黄 |
| 次要 | `#7a7aaa` |
| 线条 | `rgba(0, 240, 255, 0.3)` 发光扫描线 |

## 间距

标准 8pt Grid，但使用倾斜/切成角度

## 动效

L3-L4 — Glitch 效果、扫描线动画、粒子系统、鼠标交互

## 特效

- Glow 发光元素 (霓虹灯效)
- 色差 (Chromatic Aberration)
- 扫描线/网格叠加
- 粒子系统 (Canvas/Three.js)
- 故障效果 (Glitch)
- Shader 扭曲效果

## 布局

- Cover: 居中标题 + 故障动效 + 霓虹边框
- Stats: 发光数字 + 扫描线背景
- Gallery: 像素/色差预览

## 禁忌

- ❌ 衬线字体（破坏科技感）
- ❌ 白色背景（赛博需要黑色）
- ❌ 毛玻璃/柔和效果（不够硬核）
- ❌ 圆角过大（用 2-4px 或锐角）
