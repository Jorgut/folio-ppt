# Editorial · 编辑风

> 杂志风格 · 内容展示

## 语感

"印刷杂志搬到屏幕上" — 图文混排、大标题、阅读体验优先。

## 参考品牌

New York Times Magazine, Monocle, The New Yorker, Kinfolk

## 字体

```
标题: Playfair Display / Cormorant Garamond (字重 700)
正文: Inter / Charter (字重 400)
元数据: JetBrains Mono / IBM Plex Mono (字重 400)
引语: Playfair Display Italic (字重 400)
```

## 颜色

| 角色 | 浅色 | 深色 |
|------|------|------|
| 背景 | `#f9f9f6` | `#1a1a1a` |
| 文字 | `#1a1a1a` | `#e8e4dc` |
| 强调 | `#a67c52` 暖金 | `#c9a87c` |
| 次要 | `#666666` | `#999080` |

## 间距

标准 8pt Grid：section padding `--sp-5` / `--sp-6`

## 动效

L2 标准 — 翻页 FadeIn + 图片 Reveal

## 特效

- Noise Overlay (纸质感)
- Gradient Mesh (丰富白背景)

## 布局

- Overlap: 图文叠加（杂志灵魂）
- Bleed: 全出血图
- Editorial: CSS 双栏正文
- Pull-quote: 突出引用

## 禁忌

- ❌ 霓虹色（破坏纸质氛围）
- ❌ WebGL 或 Shader（不自然）
- ❌ 多主题色混搭
