# Interaction Engine

> 交互模式选择与组合规则

完整交互模式库见 `design-intelligence/interaction-patterns/` — 本文件为引擎级决策指南。

## 交互层级

| 层级 | 描述 | 包含 | 适用场景 |
|------|------|------|---------|
| **L0 · 静态** | 无动效无交互 | 纯翻页 | 印刷预览、PDF |
| **L1 · 轻量** | 基础翻页 + URL hash | 翻页导航 | 内部文档 |
| **L2 · 标准** | 过渡动画 + 键盘快捷键 + Lightbox | Fade/Scale 过渡 | 大部分路演 |
| **L3 · 丰富** | L2 + 视差滚动 + 滚动触发 + 卡片 Hover | ScrollReveal + Stagger + Parallax | 作品集、营销 |
| **L4 · 沉浸** | L3 + WebGL + 粒子 + 音频 + 鼠标追踪 | Three.js + Shader + Audio + Cursor | 创意展示、游戏 |

## 决策规则

```
IF 受众是高管/客户 → 使用 L2（专业但不花哨）
IF 正式/学术汇报 → 使用 L1-L2（内容为王）
IF 创意/设计作品集 → 使用 L3-L4（体验展示实力）
IF 数据 Dashboard → L2 + 图表动画（数据驱动）
IF 内部文档 → L1（最低开销）
```

## 内置交互系统

Folio 模板已内置：

| 交互 | 状态 | 说明 |
|------|------|------|
| 键盘导航 (←→↑↓Space) | ✅ 内置 | 全 deck 有效 |
| URL Hash 定位 (#3) | ✅ 内置 | 支持深链接 |
| 全屏 (F) | ✅ 内置 | 原生 Fullscreen API |
| 自动播放 (P) | ✅ 内置 | 可配置间隔 |
| 缩略图概览 (G) | ✅ 内置 | 网格跳转 |
| Lightbox | ✅ 内置 | 点击画廊图片 |
| 低功耗模式 (B) | ✅ 内置 | 关闭动效+特效 |
| 快捷键面板 (?) | ✅ 内置 | 交互式面板 |
| 滚轮翻页 | ✅ 内置 | 触摸板友好 |

## 应避免的交互

- ❌ 自动播放视频/音频（用户反感）
- ❌ 强制等待动画（用户失去控制感）
- ❌ 鼠标追踪后不恢复（干扰阅读）
- ❌ 不关闭状态（如 Lightbox 一直开着）
- ❌ 触屏 swipe 冲突（与滚动冲突）
