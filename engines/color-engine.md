# Color Engine

> 配色系统与主题选择

## 核心原则

1. **一个 deck 一套主题色** — 不中途换色
2. **60-30-10 法则** — 主色 60% / 辅色 30% / 强调色 10%
3. **对比度 ≥ 4.5:1** — WCAG AA 标准（正文）
4. **语义色保持** — 红色始终表示警告/错误，绿色表示成功

## 主题变量结构

```css
.theme-{name} {
  --page: ...;     /* 页面背景 */
  --ink: ...;      /* 主要文字 */
  --accent: ...;   /* 强调色 */
  --grey: ...;     /* 次要文字 */
  --page-inv: ...; /* 深色模式下翻转背景 */
  --ink-inv: ...;  /* 深色模式翻转文字 */
  --accent-dim: ...; /* 强调色柔和版 */
  --ink-soft: ...;   /* 柔和文字 */
  --hr: ...;         /* 分割线 */
}
```

## 决策规则

```
IF 企业/金融 → Mono (单色) 或默认
IF 科技/SaaS → Indigo 或 Ocean
IF 创意/游戏 → Neon 或 Rose
IF 自然/文化 → Forest 或 Sand
IF 时尚/品牌 → Rose 或默认(暖色)
IF 学术/研究 → Indigo 或默认
```

## 强调色使用规则

- 仅用于: CTA 按钮、重要数字、关键引用、链接
- 禁用: 正文、背景大面积、标题默认色
- 强调色面积 ≤ 页面的 10%

## 全局主题切换

`<body class="theme-indigo">` — 在模板 index.html 的 body 标签上切换。
