# Folio — Checklist

## P0（必须过）

- [ ] 大标题使用衬线字体（Playfair Display / Noto Serif SC）
- [ ] 正文使用非衬线字体（Inter）
- [ ] 元数据使用等宽字体（JetBrains Mono）
- [ ] 图片只用标准比例（21:9 / 16:10 / 4:3 / 3:2 / 1:1）
- [ ] 没有 emoji（用 Lucide 图标代替）
- [ ] 一份 deck 只使用一套主题色
- [ ] 连续不超过 3 页同主题（light/dark/hero 交替）
- [ ] HTML 在浏览器打开完全正常
- [ ] 所有图片路径正确

## P1（建议过）

- [ ] 标题与正文对比 ≥ 6:1
- [ ] 不对称 split 替代均分 50/50
- [ ] 至少有一页图文重叠布局
- [ ] 至少有一页全出血布局
- [ ] 每页有语义化入场动效
- [ ] 导航点显示正确页码

## P2（质量）

- [ ] 图片没有拉伸变形
- [ ] 文字没有溢出容器
- [ ] 留白充分，不拥挤
- [ ] 手机端布局正常

## P3（加分）

- [ ] 导出 PPTX 后检查文字可编辑
- [ ] PPTX 内字体嵌入正确
- [ ] 低功耗模式（按 B）可关闭动画
